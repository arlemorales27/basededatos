SQL - Lenguaje de Consulta Estructurado

Sentencia SELECT

La sentencia SELECT se utiliza para recuperar datos de una o más tablas en una base de datos. Es la consulta más común en SQL.

Ejemplo 1: Selección básica

SELECT nombre, apellido FROM empleados;

Este ejemplo selecciona el nombre y apellido de todos los empleados en la tabla.

Ejemplo 2: Filtrado con WHERE

SELECT * FROM productos WHERE precio > 100;

Selecciona todos los productos con un precio mayor a 100.

Ejemplo 3: Ordenamiento

SELECT nombre, precio FROM productos ORDER BY precio DESC;

Ordena los productos por precio de mayor a menor.

Ejemplo 4: Agrupación

SELECT categoria, COUNT(*) as total FROM productos GROUP BY categoria;

Cuenta cuántos productos hay en cada categoría.

Ejemplo 5: Joins

SELECT p.nombre, c.nombre_categoria FROM productos p INNER JOIN categorias c ON p.categoria_id = c.id;

Une la tabla de productos con la de categorías para mostrar el nombre del producto y su categoría.

Sentencia INSERT

La sentencia INSERT se utiliza para agregar nuevos registros a una tabla en la base de datos.

Ejemplo 1: Inserción básica

INSERT INTO empleados (nombre, apellido, edad) VALUES ('Juan', 'Pérez', 30);

Inserta un nuevo empleado con nombre, apellido y edad.
